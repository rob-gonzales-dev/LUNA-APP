# Luna Daily

Luna Daily is a free, privacy-first daily check-in app under The Human Question project.

It helps people track routines, cannabis use, alcohol use, vitamins, workouts, weight, notes, and custom habits without shame or paywalls. It supports harm reduction, self-awareness, reduction, abstinence, and ordinary daily routine-building.

## Current Version

This is a static web app:

- `index.html` provides the app structure.
- `styles.css` provides the phone-friendly design.
- `app.js` stores data locally in the browser using `localStorage`.
- `manifest.webmanifest` provides basic Home Screen metadata.
- `assets/` contains prototype visual assets.

## Features

- First-run setup with name and Luna's Default.
- Daily tracking for vitamins, workout, cannabis hits, drinks, weight, and notes.
- Custom yes/no, number, and decimal trackers.
- Sunday-to-Saturday weekly analysis.
- Gentle weekly insight language.
- Recent day markers.
- Photo uploads for personal motivation.
- JSON export/import backups.
- Supportive daily advice.

## Privacy Note

Version 1 stores data only in the browser on the device being used. There is no account, backend, sync, or cloud storage yet.
