# Luna Daily - How To Test Version 1

Open this file on Rob's computer:

`C:\Users\rgonzales\OneDrive\5. Health and VA\6. Health App\luna-daily\index.html`

What to try first:

1. Enter your name and choose `Luna's default`.
2. Check vitamins or workout.
3. Add cannabis hits, drinks, weight, and a note.
4. Add a custom tracker under `My extras`.
5. Tap the previous and next day buttons.
6. Check that weekly analysis uses Sunday through Saturday.
7. Use `Export backup` to download a JSON backup.
8. Use `Import backup` with that JSON file to confirm the app can restore data.

Current storage note:

The app stores data in the browser on the device you use. It does not yet sync across phones, computers, Google Drive, or GitHub.

iPhone note:

For a true iPhone Home Screen test, the app should eventually be hosted on a simple web address, then opened in Safari and added with Share -> Add to Home Screen. The local desktop version is the first working prototype.
